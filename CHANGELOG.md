# Changelog

Semua perubahan yang signifikan pada proyek ini akan didokumentasikan di file ini.

Format pencatatan berdasarkan [Keep a Changelog](https://keepachangelog.com/id/1.0.0/), dan proyek ini menganut [Semantic Versioning](https://semver.org/lang/id/).

## [1.0.1] - 2026-04-23

### Ditambahkan
- **Workspace Rules**: Menambahkan panduan aturan pengembangan dan manajemen versi pada proyek ini di dalam `README.md`.
- **Changelog Track**: Pembuatan file `CHANGELOG.md` untuk melacak seluruh pembaruan dan histori versi agar lebih terstruktur.
- **NPM Script `serve`**: Menambahkan script `"serve"` pada `package.json` untuk menjalankan bot dengan environment Node.

### Diperbarui
- **Peningkatan Versi**: Pembaruan sistem versi proyek ke v1.0.1 dari v1.0.0 sebelumnya.
